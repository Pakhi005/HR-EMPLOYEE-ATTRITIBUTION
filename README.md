# 🚀 Employee Attrition Predictor

An AI-powered Employee Attrition Prediction platform that helps HR teams identify employees who are at risk of leaving the organization. The application uses Machine Learning to analyze employee-related factors and predict attrition risk in real time, while providing explainable insights and actionable recommendations.

---

## 📌 Overview

Employee attrition is a major challenge for organizations, leading to increased hiring costs and productivity loss. This project predicts whether an employee is likely to leave the company based on various HR attributes.

The application features an interactive dashboard where users can enter employee details and instantly receive:

- ✅ Attrition Risk Prediction
- 📊 Risk Score
- 📈 Feature Importance
- 💡 AI-generated Recommendations
- 📉 HR Analytics Dashboard

## ⚙ Installation

```bash

```


---

### Backend Setup

```bash
pip install -r requirements.txt
```

Run Backend

```bash
python app.py
```

---

### Frontend Setup

```bash
cd frontend
```

